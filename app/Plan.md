# План тестирования мобильного приложения «Мобильный хоспис»

## 1. Границы тестирования (Scope)
* Форма авторизации (логин/пароль).
* Главный экран (список новостей).
* Панель управления новостями (создание, редактирование, удаление).
* Фильтрация новостей (по датам, по категориям).
* Раздел с цитатами («Тематические цитаты»).
* Экран «О приложении».

## 2. Типы тестирования
* **Функциональное:** проверка основных сценариев (CRUD новостей, Login).
* **UI/UX:** проверка корректности отображения элементов и навигации.
* **Negative testing:** проверка обработки неверного логина/пароля.
* **Автоматизированное:** покрытие критических сценариев (Smoke тесты).

## 3. Инструментарий
* **IDE:** Android Studio.
* **Framework:** Espresso / UI Automator.
* **Report:** Allure.
